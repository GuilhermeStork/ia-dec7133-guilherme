## Resumo do projeto em uma frase

> Usamos **K-Means** (não-supervisionado) para descobrir que existem 3 perfis de clientes, depois usamos esses perfis como rótulos para treinar uma **Árvore de Decisão** (supervisionado) que classifica novos clientes automaticamente.
